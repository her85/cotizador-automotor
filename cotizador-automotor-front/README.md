# Cotizador Automotor — Frontend

Simulador de cotización de seguros automotor. Wizard de 4 pasos en Vue 3 + TypeScript.

## Instalación

```bash
npm install && npm run dev
```

## Pasos del wizard

1. **Vehículo** — Marca → Modelo (carga dinámica) → Año (filtrado por modelo)
2. **Conductor** — Edad, zona, uso, historial de siniestros
3. **Cobertura** — Selección visual con coberturas y sus incluidos
4. **Resultado** — Prima mensual/anual, desglose de factores, guardar cotización

## Stack

Vue 3 + TypeScript · Vite · Tailwind CSS 3 · Pinia · Vue Router 4 · Axios
